    Project Info

    • Authors: Felipe Ortega, Aaron Halfaker.
    • License: GPLv3.

    The aim of WikiDAT is to create an extensible toolkit for Wikipedia Data Analysis, based on Python and R. Currently, MySQL is the only supported DB backend, but there are plans to include other DB engines in the future.

    Each module implements a different type of analysis and it stores the output in subdirectories results, figs or traces, inside the module's maind folder. Every module includes Python and R code to implement both the data preparation, cleaning and analysis steps, including inline comments. An important goal is to illustrate different case examples of interesting analyses with Wikipedia data, following a didactic approach.

    The long-term goal is to include more case examples progressively, in order to cover many of the usual examples of quantitative analyses that can be implemented on Wikipedia data. In the future, this may also include the use of tools for distributed computing to support analysis of really huge data sets in high-resolution studies.

    Required dependencies The following software dependencies are required to run all examples currently included in WikiDAT:

    • MySQL server and client (>= v5.5).
    • Python programming language (>= v2.7 bu < v3) and Python MySQLdb (>= v1.2.3)
    • R project for statistical computing (>= v2.15.0).

    Additional R packages with extra data and functionalities (This list will be updated as new functionalities are included):

    • RMySQL: Connect to MySQL dbs from R.
    • Hmisc: Frank Harrell's miscelaneous functions (essential).
    • car: Companion package for "R Companion to Applied Regression", 2nd ed.
    • DAAG: Companion package for "Data Analysis and Graphics using R."
    • ineq: Calcualte inequality metrics and graphics.
    • ggplot2: A wonderful package to create appealing graphics in R.
    • eha: Package for event history and survival analysis.
    • zoo: Excellent library to handle timeseries data.
